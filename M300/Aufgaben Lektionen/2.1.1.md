# 2.1.1 – Infrastruktur aufbauen (Vorbereitung)

## Legende
- LS = Linux Server
- LF = Firewall

---

## 1. Firewall testen (LF)
ping www.gbssg.ch

---

## 2. Login Linux VM (LS)
User: vmadmin  
Passwort: Riethuesli>12345

---

## 3. root aktivieren (LS)
sudo passwd root

Passwort setzen:
Riethuesli>12345

---

## 4. Neustart (LS)
reboot

→ danach als root einloggen

---

## 5. Netzwerkkarte einrichten (LS)

ip addr

cp /etc/netplan/*.yaml /home/vmadmin/

mv /etc/netplan/*.yaml /etc/netplan/config.yaml

nano /etc/netplan/config.yaml

### Inhalt:
network:
 version: 2
 renderer: networkd
 ethernets:
  ens33:
   addresses:
    - 192.168.210.41/24
   routes:
    - to: default
      via: 192.168.210.1
      metric: 100
      on-link: true
   nameservers:
    search: [<deinname>.m300]
    addresses: [192.168.210.41,192.168.210.1]

netplan apply

---

## 6. Verbindung testen (LS)
